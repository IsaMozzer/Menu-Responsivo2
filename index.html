O que é Flexbox?

Flexbox é uma forma do CSS organizar elementos em linha ou coluna de maneira flexível, permitindo alinhar e distribuir o espaço entre eles automaticamente, fazer com que cresçam ou encolham conforme o tamanho da tela, sendo muito usado para criar layouts responsivos de forma simples.


Quando usar Flexbox?

Você usa Flexbox quando quer organizar elementos de forma alinhada e adaptável, como:


  • Criar menus, barras de navegação ou rodapés.
  • Centralizar um item no meio da tela, na horizontal e/ou vertical.
  • Distribuir espaço entre elementos sem calcular manualmente margens.
  • Fazer um layout que se ajuste bem em diferentes tamanhos de tela.

Resumindo: use quando precisar alinhar e distribuir elementos de forma prática e responsiva.


Por que usar Flexbox?

Usar Flexbox vale a pena porque ele facilita muito o alinhamento e a organização dos elementos, reduz a quantidade de código, funciona bem em layouts responsivos e evita cálculos manuais de tamanhos e espaçamentos, tornando o desenvolvimento mais rápido e simples.

Bonus

  • Ordem flexível: Você pode mudar a ordem dos elementos na tela sem alterar o HTML, usando apenas a propriedade order
  • Centralização perfeita: Com poucas linhas, você consegue centralizar qualquer elemento na vertical e horizontal, algo que antes era bem trabalhoso no CSS.
  • Espaço automático: O Flexbox consegue distribuir espaço extra entre itens (space-between, space-around, space-evenly) sem precisar de margens manuais.
  • Mistura com outros layouts: Ele funciona muito bem junto com CSS Grid, combinando flexibilidade com controle preciso de áreas.
